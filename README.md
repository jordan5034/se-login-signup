# se-login-signup

A simple login and signup web application built with HTML and CSS.

## Project Structure

- `login.html` - Login page with User ID and Password fields
- `signup.html` - Signup page with First Name, Last Name, User ID, and Password fields
- `styles.css` - Shared stylesheet used by both pages
- `README.md` - Project documentation

## How to Run

1. Download or clone the repository
2. Open `login.html` in a web browser

## Pages

**Login** — allows an existing user to sign in with their User ID and password.

**Sign Up** — allows a new user to register with their first name, last name, user ID, and password.
